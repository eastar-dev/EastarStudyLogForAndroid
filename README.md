## What is Eastar StudyLog for Android
현재 지행하는 스터디 그룹에 스터디에 도움이 되고 서로서로 자극이 되어 좀더 높은 효율을 낼수 있도록 만든 엡

## How to use
1. 회원가입
+ 닉네임 입력
+ 스터디 그룹 입력
2. 스터디 활동 작성
